# AWENUE Hero Video Assets

Place the production video files here before deployment.

## Required Files

| File | Format | Size Target | Notes |
|------|--------|------------|-------|
| `awenue-hero.webm` | WebM (VP9) | 4–8 MB | Primary format, best compression |
| `awenue-hero.mp4`  | MP4 (H.264) | 6–10 MB | Universal fallback |
| `awenue-hero-mobile.mp4` | MP4 (H.264) | 2–4 MB | Optional smaller version |

## Video Specifications

- **Duration**: 12–20 seconds, seamlessly loopable
- **Aspect Ratio**: 16:9
- **Resolution**: 1920×1080 (desktop), 1280×720 (mobile)
- **Frame Rate**: 24–30 fps
- **Audio**: No audio track required
- **Main visual action**: Center-right of frame (left side reserved for headline text)
